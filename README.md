Autor projekta: \

Ukratko o projektu: \
Projekat predstavlja scenu lika iz crtanog filma Sundjer Boba Kockalone, njegovog puza Garija ispred njihove kuce gde lete meduze, a autobus prolazi..

## Kontrole

- `W`, `A`, `S`, `D` - kretanje po sceni
- `F1` - otvara ImGui
- `B` - kretanje/zaustavljanje autobusa
- `L` - smena day/night scena
- `ESC` - izlazak iz aplikacije
