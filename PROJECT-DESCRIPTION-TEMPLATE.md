# Forest ambience

287/2018 - Dejan Zivkovic  
Calm scene with a forest ambience that contains a spooky mansion, street lamp and a bunch of trees.

## Controls

F2 -> Open/Close GUI  
WASD -> Camera movement  
Arrow keys -> Camera movement
Mouse moving -> Rotate camera

1 -> Change color of the Moon (white)
2 -> Change color of the Moon (red)
3 -> Change color of the light cubes (bulbs) on the street lamp (yellow)
4 -> Change color of the light cubes (bulbs) on the street lamp (white)
5 -> Change color of the light cubes (bulbs) on the street lamp/turning them off (black)

## Features

### Fundamental:

[X] Model with lighting

[X] Two types of lighting with customizable colors and movement through GUI or ACTIONS

[X] Pressing anything in [1,2,3,4,5] will change the color of the lighting at the scene

### Group A:

[ ] Frame-buffers with post-processing   
[X] Instancing  
[ ] Off-screen Anti-Aliasing  
[ ] Parallax Mapping

### Group B:

[ ] Bloom with the use of HDR  
[ ] Deferred Shading  
[ ] Point Shadows  
[ ] SSAO

### Engine improvement:

[X] Added new methods in engine/resources/Mesh.hpp and engine/resources/Model.hpp for instancing
