# Cottage

135/2022 - Bojana Zagorac
{Cottage with some trees on the field}

## Controls

W/A/S/D -> forward/left/backward/right  
UP/DOWN Arrow -> Increase/Decrease light intensity
X -> Change directional light to red for 6 seconds
F2 -> Open GUI
ESC -> Exit


## Features
Models: Cottage,3 Tree models

### Fundamental:


[+] Model with lighting
[+] Two types of lighting with customizable colors and movement through GUI or ACTIONS
[+] X button  -> AFTER 0 SECONDS -> Triggers -> Directional Light color change to red -> AFTER 6 SECONDS -> Triggers -> Returns to normal white light

### Group A:

[ ] Frame-buffers with post-processing   
[ ] Instancing  
[ ] Off-screen Anti-Aliasing  
[ ] Parallax Mapping

### Group B:
[ ] Bloom with the use of HDR  
[ ] Deferred Shading  
[ ] Point Shadows  
[ ] SSAO
