# Weird dream

130/22 - ZELJKO ZEKAVICIC
Scene in the sky, where everything is paranormal.
Some kind of dream-antidream.

## Controls

WASD -> Forward, backward, left, right
Left Shift -> Down
Space -> Up
L -> Turn on/off light on lamp by the bedside table
B -> Turn on/off light bloom effect
X -> "Morbin time", switch lamp light to red and make clouds move anti-clockwise
F2 -> Turn GUI on/off with 50/50 button

## Features

You can try your luck with flickering lights via GUI,
or with going back in time (turning clouds anti-clockwise and lights red).
Foggy effect with bloom that can easily be turnt off/on by pressing "B".
