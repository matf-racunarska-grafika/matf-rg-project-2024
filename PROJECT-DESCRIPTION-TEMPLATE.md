# Melancholic street

13/2022 - Nikola TrifunoviÄ‡  
Melancholic street - A 3D simulation of a street with 10 buildings, post lamps and a road

## Controls
W - Move forwards\
CTRL+W - Move forwards faster\
A - Move left\
S - Move backwards\
D - Move right\
SPACE - Move up\
SHIFT - Move down\
P - Turn all lights off\
J - Start "light show", direction left\
L - Start "light show", direction right\
K - Stop "light show"\
B - Open GUI

## Features

### Fundamental:

[+] Model with lighting\
[+] Two types of lighting with customizable colors and movement through GUI or ACTIONS\
[+] {Key L is pressed} --- AFTER_0.5_SECONDS---Triggers---> {Lights rotate colors to the right} --->
AFTER_0.5_SECONDS---Triggers---> {Lights rotate colors to the right} ---> AFTER_0.5_SECONDS---Triggers---> ...

### Group A:

[ ] Frame-buffers with post-processing   
[+] Instancing  
[ ] Off-screen Anti-Aliasing  
[ ] Parallax Mapping

### Group B:
[+] Bloom with the use of HDR  
[ ] Deferred Shading  
[ ] Point Shadows  
[ ] SSAO

### Engine improvement:

[+] Added "Drawable" class for easier work with models
